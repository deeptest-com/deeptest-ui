## 软件测试平台 - 前端

### 快速开始

1.执行命令，初始化项目

```
# 拉取前端的 配置信息
git clone git@gitlab.nancalcloud.com:leyanapi/frontend-config.git

sh ./init.project.sh
``` 

2. 安装依赖

- node为 v16.x.x
- yarn为 v1.22.xx

```
yarn install
```

3. 启动项目

```
yarn serve
```
